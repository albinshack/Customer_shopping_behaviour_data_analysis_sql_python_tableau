# Customer Shopping Behaviour Data Analysis

## Project Overview
This project analyzes customer shopping behaviour to identify purchasing trends, customer segments, and product performance. The analysis combines SQL for data querying, Python for data processing and exploration, and Tableau for interactive visualizations.

The goal of this project is to demonstrate a complete data analytics workflow from raw data extraction to visualization and insights.

## Tools & Technologies
- SQL
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Tableau
- Data Visualization

## Project Workflow

### 1. Data Collection
The dataset contains customer transaction records including purchase amounts, product categories, and shopping trends.

### 2. Data Exploration
Using Python and Pandas, the dataset was cleaned and explored to identify patterns, missing values, and distributions.

### 3. SQL Analysis
SQL queries were used to extract insights such as:
- Top selling product categories
- Customer purchase frequency
- Revenue by category
- Monthly sales trends

### 4. Visualization
Tableau dashboards were created to present key insights interactively, including:
- Sales distribution
- Customer segmentation
- Category performance
- Shopping trends

## Key Insights
- Certain product categories dominate overall sales.
- Customer purchasing patterns vary significantly by category.
- Seasonal trends influence sales performance.
- High-value customers contribute a large portion of revenue.

## Project Files
- `sql/analysis_queries.sql` – SQL queries used for analysis
- `notebooks/data_analysis.ipynb` – Python exploratory data analysis
- `tableau/dashboard.twbx` – Tableau dashboard file
- `images/` – Dashboard screenshots

## Tableau Dashboard
Dashboard visualizes sales performance, customer behaviour, and product trends.

## How to Run the Project
1. Clone the repository
2. Open the Jupyter notebook in the `notebooks` folder
3. Run the SQL queries in your SQL environment
4. Open the Tableau workbook to explore the dashboards
